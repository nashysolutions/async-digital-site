# async-digital-site

Marketing site for Async Digital, hosted at [async-digital.com](https://async-digital.com) via GitHub Pages.

## Structure

- `index.html` — landing page
- `styles.css` — shared styles
- `assets/` — logos, favicons, Open Graph images
- `four-blocker/` — internal one-pager
- `CNAME`, `robots.txt`, `sitemap.xml` — hosting and SEO config

## Editing

Commit changes to `main` — GitHub Pages deploys automatically.
